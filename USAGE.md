# 使用说明

## 启动应用

开发环境：
```bash
npm run dev
```

访问 `http://localhost:5173` 查看主界面。

## 主界面

主界面显示 "SVAnimation 短视频特效助手" 标题，下方列出所有可用的特效模块卡片。

- 点击任意模块卡片进入该模块
- 每个模块都有独立的功能界面
- 点击顶部的"返回主界面"按钮可返回模块选择页面

## 倒鸭子字幕模块

### 基本操作

1. **导入字幕**：点击"导入 SRT"按钮，选择 `.srt` 字幕文件
2. **导入音频**：点击"导入 MP3"按钮，选择 `.mp3` 音频文件
3. **播放预览**：使用播放器控件预览动画效果
4. **恢复演示**：点击"恢复中文演示字幕"加载默认演示内容

### 配置选项

#### 输出分辨率
- 1920 x 1080 (Full HD)
- 1280 x 720 (HD) - 默认
- 1080 x 1080 (Square)
- 1080 x 1920 (Vertical)
- 2560 x 1440 (2K)

#### 中心区域框选
1. 点击"进入框选模式"按钮
2. 在播放器中拖拽鼠标框选中心区域
3. 松开鼠标自动应用并退出框选模式
4. 可勾选"显示中心区域(黄框)"查看约束范围

#### 字幕进入动画
可多选以下动画：
- 砸入弹跳
- 旋转缩放
- 侧滑扭曲
- 弹出抖动
- 弹跳摇摆

多选时，每条新字幕会随机使用其中一种动画。

#### 中心区域精准数值
使用数值输入框精确设置中心区域：
- X 位置 (%)
- Y 位置 (%)
- 宽度 (%)
- 高度 (%)

### 动画规则

1. **整句显示**：中文字幕以整句为单位显示，不拆字不拆词
2. **几何贴合**：新字幕的左上角贴合上一条字幕的左下角
3. **分组旋转**：每3条字幕为一组，完成后向左旋转90度
4. **自动回中**：字幕会自动平移以保持在中心区域内
5. **淡出过渡**：旧的旋转块会在新旋转开始前淡出

## 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

构建产物为单文件 HTML，位于 `dist/index.html`，可直接在浏览器中打开使用。

## 文件格式要求

### SRT 字幕文件

标准 SRT 格式：
```
1
00:00:00,000 --> 00:00:01,200
第一条字幕内容

2
00:00:01,300 --> 00:00:02,400
第二条字幕内容
```

### MP3 音频文件

- 格式：MP3 (MPEG Audio Layer 3)
- 建议比特率：128kbps 或更高
- 支持任意采样率

## 技巧和注意事项

1. **同步音频**：确保 SRT 字幕时间轴与 MP3 音频对应
2. **字幕数量**：建议每次使用 9-15 条字幕（3-5 组）效果最佳
3. **中心区域**：默认中心区域为画面中央 44% x 42%，可根据需要调整
4. **分辨率选择**：竖屏视频选择 1080x1920，横屏选择 1920x1080
5. **动画选择**：可以只选择 1-2 种喜欢的动画保持风格一致
6. **浏览器兼容**：推荐使用 Chrome、Edge 或 Safari 最新版本

## 导出视频

本工具提供实时预览功能。如需导出为视频文件，可以：

1. **屏幕录制**：使用 OBS、QuickTime 等工具录制播放器画面
2. **Remotion CLI**：使用 Remotion 官方 CLI 工具渲染（需要额外配置）

## 故障排除

**字幕不显示**
- 检查 SRT 文件格式是否正确
- 确认时间轴格式为 `HH:MM:SS,mmm`

**音频无法播放**
- 确认文件为 MP3 格式
- 检查浏览器是否支持音频自动播放

**动画卡顿**
- 尝试降低分辨率
- 关闭浏览器其他占用资源的标签页

**框选不生效**
- 确保已点击"进入框选模式"按钮
- 框选区域需要大于 8x8 像素才会生效
