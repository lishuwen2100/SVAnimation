# Bug 修复总结 - 字幕样式配置丢失

## 问题描述

在将 DuckSubtitle 模块拆分为工作流节点时，字幕样式配置功能被遗漏：
- ❌ 字幕位置设定（随机/手动模式）

- ❌ 单条字幕编辑（字体、字号、动画、位置）
- ❌ 批量操作（全选/清空/重置）
