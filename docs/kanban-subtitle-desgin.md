# Kanban Subtitle

## 整体样式的改变
- 把上传、分辨率编辑模块调整到左侧
- 分辨率模块可以收起和展开，默认收起

## 添加字幕模块

- 字幕模块有和倒鸭子的字幕样式设计中一样的字幕列表
- 字幕设定模块也是可以展开和收起的，默认收起
- 点击添加后就在字幕列表中新增一条字幕
- 字幕的样式
  - 进入画面的时间点 （获取当前视频的时间戳作为新增字幕的进入时间点）
  - 移出画面的时间点 （默认为0，即一直显示，由按钮可以获取当前视频时间戳作为移出画面的时间点）
  - 字体大小
  - 颜色（默认白色）
  - 坐标（默认视频左上角，可以由用户通过鼠标点选且可以通过数值填写进行微调）
  - 进入动画（参考倒鸭子的动画种类，默认为没有进入动画，直接按照前面的设定显示出字幕）
  - 缩放 （开关，如果打开则有下面的设定）
    - 字体大小
    - 坐标 （默认跟当前字幕坐标一致，可以由用户通过鼠标点选且可以通过数值填写进行微调）
    - 缩放间隔
    - 缩放功能打开后，按照时间间隔等待，时间到了之后，让字幕从当前坐标向缩放设定的坐标进行缩放的动画变幻，且把缩放的坐标、字体作为字幕后续的生效设定
  - 字幕增加后，播放视频时字幕会按照设定显示、进入动画、缩放动画或者消失
