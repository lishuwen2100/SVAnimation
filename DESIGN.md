# 倒鸭子字幕播放器设计说明

## 1. 项目目标
- 使用 Remotion Player 预览字幕动画。
- 支持导入 SRT 字幕与 MP3 音频。
- 以中文整句为单位渲染字幕，遵循“倒鸭子/Typemonkey”风格：贴合、分组、旋转、回中。

## 2. 技术与结构
- 技术栈: React + Vite + Tailwind CSS + Remotion。
- 入口组件: `src/App.tsx`。
- 文档: `DESIGN.md`。

## 3. 页面布局
- 左侧主区:
  - 标题与说明。
  - SRT / MP3 导入按钮，恢复演示字幕按钮，字幕数量与时长信息。
  - Remotion Player 播放器。
- 右侧配置区:
  - 每个功能以折叠模块（`details`）组织，按需展开配置。
  - 当前模块:
    - 输出分辨率
    - 中心区域框选
    - 字幕进入动画
    - 中心区域精准数值

## 4. 输入与时间轴
- SRT 解析:
  - 读取 `startSec/endSec` 与整句文本。
  - 不拆词不拆字，按整条字幕显示。
  - 按时间升序。
- 帧参数:
  - `FPS = 30`
  - 每 3 条为一组。
  - 组切换时加入旋转过渡时间补偿。

## 5. 字幕几何贴合逻辑
- 第一条字幕: 落在中心区域附近稳定随机位置（基于文本哈希）。
- 同组贴合:
  - 新字幕的 `top-left` 贴合上一条字幕的 `bottom-left`。
- 组旋转:
  - 每 3 条触发一次。
  - 旋转圆心为该组外接矩形的 `bottom-left`。
  - 方向为向左 90 度。
- 旋转后新组起点:
  - 新组第一条用自身 `bottom-left` 贴合上一旋转组外接矩形 `bottom-right`。

## 6. 旋转时序规则
- 当新一组准备旋转时:
  - 若已存在“上一轮旋转字幕块”，先淡出消失。
  - 消失到阈值后，再开始当前组旋转。
- 旋转期间避免重复显示同组静态字幕与旋转字幕，防止双影。

## 7. 中心区域约束
- 定义中心区域矩形: `x, y, width, height`（归一化比例）。
- 每条新字幕落点后检查其中心点是否在中心区域内。
- 如果不在，对当前整体字幕应用一次平移动画（累积偏移），将其推回中心区域边界内。
- 平移不改变字幕之间的贴合拓扑。

## 8. 中心区域配置
- 默认显示黄色中心区域框。
- 鼠标框选:
  - 在左侧播放器进入框选模式后可拖拽选区。
  - 松开后更新中心区域并自动退出框选模式。
- 精准设置:
  - 在折叠面板中输入 X/Y/宽/高百分比。
  - 自动边界裁剪，防止越界。

## 9. 字幕进入动画
- 支持勾选动画（可多选）:
  - 砸入弹跳
  - 旋转缩放
  - 侧滑扭曲
  - 弹出抖动
  - 弹跳摇摆
- 选择规则:
  - 多选时，每条新字幕从勾选项中稳定随机选一种。
  - 全不选时使用默认入场动画。
- 约束:
  - 动画只影响进场过程的 `transform/opacity/blur`。
  - 最终落点始终服从贴合几何规则。

## 10. 分辨率模块
- 右侧提供单选分辨率配置（主流预设）:
  - 1920x1080 (Full HD)
  - 1280x720 (HD)
  - 1080x1080 (Square)
  - 1080x1920 (Vertical)
  - 2560x1440 (2K)
- 选择后同步影响:
  - `compositionWidth / compositionHeight`
  - Player 展示纵横比
  - 字幕布局计算与中心区域约束坐标系

## 11. 关键状态
- 媒体与字幕:
  - `srtText`
  - `audioSrc`
  - `audioDuration`
  - `cues`
- 画布与控制:
  - `resolutionId`
  - `centerRegion`
  - `isDrawMode`
  - `draftRect`
  - `selectedAnimations`
